# MYLStockDemo
### 描述 
一个在别的第三方基础上完成的，包含分时图(五档明细图)、五日图、k线图（蜡烛图） 的项目，可横屏 、十字线拖动 ma5 、ma10、ma20 vol macd kdj rsi 等计算和绘制，基本满足股票类app 页面的实现
